Software Projects

Current

  • Nonstd: Nano-library providing very few Core-like modules (like List, Option) (see on Bitbucket).
  • Sosa: The “Sane OCaml String API” library is a set of APIs (module types) that define what a string of characters should be, and a set of modules and functors that implement them (see on Github, and also the documentation for version 0.0.1).
  • Docout: The functor Docout.Make_logger creates a nice embedded DSL on top of the smart-print library (see on Bitbucket).
  • Pvem: The "Polymorphic Variants-based Error Monad", Pvem (pronounce /pi:vɛm/), is a module providing simple handling of an error monad type based on polymorphic variants (see on Bitbucket).
  • Atd2cconv: The atd2cconv application compiles ATD descriptions to OCaml code which defines CConv sources and sinks (see on Github, and also the documentation for version 0.0.0).
  • Pvem_lwt_unix: Pvem_lwt_unix provides a high-level API on top of Lwt_unix, with comprehensive error types (see on Bitbucket).
  • Ketrew: Workflow Engine for complex computational experiments (see on Github, and also the documentation for version 0.0.0).
  • Oredoc: Build documentation websites for some OCaml projects (see on Github).
  • Trakeva: Transactions, Keys, and Values: an API describing key-value storage with ACID transactions and different backends (see on Github, and also the documentation for version 0.0.0).
