NAME

ketrew−explore − Run the interactive Target Explorer.

SYNOPSIS

ketrew explore [OPTION]...

COMMON OPTIONS

−C FILE, −−configuration−file=FILE
(absent=_test_env/server−config−file.toml)

Use FILE as configuration file (can be overriden also with ‘KETREW_CONFIGURATION‘).

−−help[=FMT] (default=pager)

Show this help in format FMT (pager, plain or groff).

−−version

Show version information.
