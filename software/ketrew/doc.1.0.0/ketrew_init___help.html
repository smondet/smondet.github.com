NAME

ketrew−initialize − Initialize the client (create a config−file)

SYNOPSIS

ketrew initialize [OPTION]... PATH

ARGUMENTS

PATH

Path of the generated config file

COMMON OPTIONS

−−help[=FMT] (default=pager)

Show this help in format FMT (pager, plain or groff).

−−version

Show version information.

OPTIONS

−−database=FILE (absent=/Users/sebastienmondet/.ketrew/database)

Use FILE as database.

−−debug−level=INT (absent=1)

Set the debug−level INT.
