NAME

ketrew − A Workflow Engine for Complex Experimental Workflows

SYNOPSIS

ketrew COMMAND ...

COMMANDS

explore

Run the interactive Target Explorer.

initialize

Initialize the client (create a config−file)

inspect

Run steps of the engine.

interact

Run the interactive menu.

kill

Kill a target.

print−configuration

Display current configuration.

run−engine

Run steps of the engine.

start−server

Start the server.

status

Get info about this instance.

stop−server

Stop the server.

OPTIONS

−−help[=FMT] (default=pager)

Show this help in format FMT (pager, plain or groff).

−−version

Show version information.

SOME COMMAND ALIASES

pc

An alias for print−configuration.
